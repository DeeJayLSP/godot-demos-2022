New features in Godot 3.5

This collection of demos highlights some of the significant new features in Godot 3.5.

We made it along with our video dedicated to The Cool New Features in Godot 3.5 (coming soon, subscribe to our channel to know when it comes out).

This project requires Godot 3.5 to open.

It showcases:

  1. The new navigation server with support for moving obstacles. The demos show it working both in 2D and 3D.
  2. The new SceneTreeTween that allows you to create tweens without creating nodes.
  3. The new Label3D node, which displays text as billboards or double-sided quads.
  4. The new flow container for user interface, which is a bit like flexbox in CSS.
  5. The physics interpolation feature, which allows you to lower the physics updates, save performance, yet get smooth motion.
  6. The 2D gradient texture you can use for lights, particle placeholders, and masking.
  7. The ability to align icons on buttons.
  8. The use of property pinning on inherited scenes.
  9. The text mesh resource to extrude 3D text.
  10. And the theme type variations that allow you to use your global project theme and make variations of anything.
